# üìä Attendance Tracker (Excel Macro-Enabled Workbook)

This project is a **Microsoft Excel-based Attendance Tracker** designed to log **check-in** and **check-out** times for individuals such as employees or students.  
It leverages **VBA macros** for automation, making attendance management faster, more accurate, and easier to maintain.

## üìÅ File Details
- **File name:** `Attandance, Check in-out.xlsm`
- **Format:** Macro-Enabled Excel Workbook
- **Created with:** Microsoft Excel (compatible with 2010 and above)
- **Primary use case:** Recording and managing attendance data efficiently.

---

## ‚ú® Features
1. **One-Click Check-In / Check-Out**
   - Buttons on the sheet log time automatically.
2. **Automatic Date Stamping**
   - No manual entry ‚Äî dates and times are filled instantly.
3. **Editable List of Names**
   - Easily add/remove employees or students.
4. **Summary Sheet**
   - Monthly total hours worked or attended.
5. **Macro-Driven**
   - Automates repetitive work without formulas.
6. **Reset Function**
   - Clear the sheet for a new entry filling in one click.
